Static html generator with Webpack 5
====================================

with Sass.

[> ORIGINAL with Webpack 4](https://github.com/ivarprudnikov/webpack-static-html-pages)
--------------------------------

You could also use this repository as a template when creating a new one.

## Versions

- `webpack: "5.10.1"`
- `webpack-cli: "4.2.0"`
- `webpack-dev-server: "3.11.0"`

## Development

- `npm install` - install dependencies
- `npm start` - start development server

## Production

- `npm run build` to prepare `html`, `css`, `js` files in `dist/` directory
- `npm run preview` - run build and serve production files locally
